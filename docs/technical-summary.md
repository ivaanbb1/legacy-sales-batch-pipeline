# Resumen tecnico

## Objetivo

Procesar ventas diarias desde ficheros planos de ancho fijo y preparar una salida validada para reporting SQL.

## Alcance

- Lectura de ficheros de ancho fijo.
- Validacion contra maestro de productos.
- Validacion de transacciones de venta.
- Generacion de fichero de ventas aceptadas.
- Generacion de fichero de ventas rechazadas.
- Informe de control del proceso batch.
- Esquema SQL y consultas analiticas.
- Especificacion de dashboard en Power BI.

## Tecnologias

- COBOL.
- Ficheros planos.
- PowerShell.
- SQL.
- Power BI.

## Posibles ampliaciones

- Validacion contra maestro de clientes.
- Historico mensual de ventas.
- Calculo de margen con costes de producto.
- Carga equivalente en SQL Server.
- Casos de prueba automatizados para entradas y salidas.
